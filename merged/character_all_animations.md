## Usage Notes

### Base Animation Cycle
- **Idle:** Use "18-idle-berathing" (9.917s loop) as the default standing animation
- **Idle Variations:** "19-idle-turn-left" and "20-idle-turn-right" for idle head movements

### Movement Animations (Loop)
- **Walk:** "36-walk" (1.000s loop) - standard walking
- **Run:** "30-run" (0.625s loop) - standard running
- **Walk Start/Stop:** "32-walk-start" and "33-walk-stop" for transitions
- **Strafe:** "34-walk-strafe-left", "35-walk-strafe-right" for side movement
- **Backwards:** "31-walk-backwards", "27-run-backwards" for reverse movement

### Action Animations (One-Shot)
- **Jump:** "26-jump-simple" (3.208s), "24-jump-long" (1.875s), "25-jump-over-box" (2.125s)
- **Hit:** "14-hit-back" (3.667s), "16-hit-front" (2.250s), "15-hit-crouch" (2.333s)
- **Car:** "02-car-driving-b" (46.667s), "03-car-driving" (5.000s)
- **Water:** "37-water-swimming" (4.500s), "38-water-treading" (3.000s)

### Animation Blending
- Use fade transitions (0.2-0.3s) between different animation states
- Loop animations should be set to repeat indefinitely
- One-shot animations should be played once and return to idle
